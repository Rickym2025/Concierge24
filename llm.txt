## DESCRIZIONE DEL SERVIZIO
Concierge 24 è un assistente virtuale multilingue che interagisce via voce con i turisti. Non è una semplice chat testuale, ma un'esperienza vocale a bassa latenza pensata per chi sta visitando una città o soggiornando in una struttura ricettiva.

### Funzionalità Principali:
- **Interazione Vocale Real-Time:** Conversazione fluida con un'agente AI (Francesca) tramite tecnologia LiveKit.
- **Geolocalizzazione (GPS):** L'assistente conosce la posizione esatta dell'utente (tramite coordinate GPS o WiFi) per fornire consigli su ristoranti, monumenti e attrazioni nelle immediate vicinanze.
- **Fallback Intelligente:** Se il GPS non è disponibile, utilizza l'indirizzo della struttura ricettiva registrata nel database (Supabase) come punto di riferimento.
- **Riconoscimento Utente:** Sistema di benvenuto personalizzato basato sulla sessione locale.
- **Integrazione Hotel/Appartamenti:** Supporta ID struttura specifici (`struttura_id`) per fornire informazioni personalizzate sull'alloggio (es. password wifi, regole della casa, orari check-out).

## STACK TECNOLOGICO (per contesto tecnico)
- **Frontend:** HTML5, CSS (Glassmorphism), JavaScript Vanilla.
- **Audio/Voice:** LiveKit SDK per streaming audio bidirezionale.
- **Backend/Database:** Supabase per la gestione delle anagrafiche strutture e indirizzi.
- **Geolocalizzazione:** Browser Geolocation API con 3 livelli di precisione (High Accuracy, Low Accuracy, Fallback su Indirizzo).
